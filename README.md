# Travel Buddy — React Navigation Demo App

**Tujuan:** Praktikum Pertemuan 9 — implementasi Stack & Tab Navigator

**Features:**
- ✅ Bottom Tab Navigation (Home, Search, Favorites)
- ✅ Stack Navigator di Home Tab (Home → Detail)
- ✅ Data passing via route params
- ✅ Icons dari @expo/vector-icons
- ✅ Fitur Manajemen Data (React Hooks)
- ✅ Fitur UI & Tata Letak (React Native Core)
- ✅ Fitur Animasi & Estetika (Visual)
- ✅ Fitur Sistem Navigasi (React Navigation)

**Cara Jalankan:**
npx expo start → Scan QR Code dengan Expo Go
